# AI-Based Fashion Stylist & Recommender System

## Overview

Welcome to the **AI-Based Fashion Stylist & Recommender System**! This project combines **fashion recommendations, a personalized AI stylist, and a weather-aware outfit recommender** into one integrated platform. The system consists of a website with multiple sections, an AI-powered recommendation engine, and a chatbot for fashion assistance.

## Features

### 1️⃣ Website

Our interactive **fashion website** provides users with multiple features:

- **Home** – Introduction to the AI stylist and recommender system.
- **Recommendations** – Personalized outfit suggestions based on user preferences.
- **Occasions** – Helps users select outfits based on formal, casual, party, or other events.
- **Weather-Based Styling** – Suggests outfits suitable for the current weather.
- **Weather App** – Integrated weather app for real-time weather updates.

### 2️⃣ AI-Powered Fashion Recommender

Our recommender system suggests outfits based on:

- **User-uploaded images** – Users can upload their current outfits for enhancement suggestions.
- **User preferences** – Users choose from categories like Formal, Casual, Party, Ethnic, or Modern styles.
- **Weather conditions** – The system considers temperature and weather type before suggesting outfits.

### 3️⃣ AI Stylist Chatbot

We have developed a **Google AI Studio-powered chatbot** that assists users with:

- Outfit selection based on gender, occasion, and style preferences.
- Styling advice and fashion trends.
- Guidance on mixing and matching outfits.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Streamlit (for UI development)
- **Backend:** Flask, Python
- **Machine Learning:** TensorFlow, Keras
- **AI Chatbot:** Google AI Studio
- **Weather API:** OpenWeatherMap API

**How It Works**

1. Users visit the website and navigate through different sections.
2. They can either upload an outfit image or input preferences manually.
3. The AI model processes the input and generates outfit recommendations.
4. Users receive **fashion recommendations** considering the occasion, weather, and personal preferences.
5. The chatbot is available for personalized **fashion advice and styling tips**.
