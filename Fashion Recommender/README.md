# AI-Based Fashion Recommender & Stylist

## 📌 Project Overview
The **AI-Based Fashion Recommender & Stylist** is a smart system that provides personalized fashion recommendations based on user preferences, weather conditions, and uploaded images. This project consists of two main components:

1. **Fashion Recommender** - Suggests outfit options based on user inputs like occasion, style preference, and weather.
2. **AI Stylist Assistant** - Analyzes uploaded images and provides feedback on styling while recommending matching clothing items.

## 🚀 Features
- **Personalized Outfit Suggestions**: Users can input their preferred fashion style, occasion, and weather details to receive tailored outfit recommendations.
- **Image-Based AI Styling**: Upload an outfit image, and the AI stylist will provide insights and complementary outfit suggestions.
- **Weather-Adapted Fashion**: Recommendations adjust based on real-time weather conditions.
- **Gender-Specific Styling**: The chatbot dynamically adapts recommendations for men and women based on input.
- **User-Friendly Chatbot**: A Google AI Studio-powered chatbot assists users with fashion advice in an interactive manner.

---

## 🛠️ Tech Stack
- **Frontend**: Streamlit (for UI)
- **Backend**: Flask (if applicable) & Python
- **Machine Learning**: TensorFlow/Keras (for image-based model)
- **Dataset**: DeepFashion-1 (for training the recommendation model)
- **Chatbot**: Google AI Studio (for interactive fashion assistant)

## 🎯 Usage Guide
### **Fashion Recommender**
1. Select the **Occasion** (Casual, Formal, Party, etc.).
2. Choose a **Style Preference** (Modern, Ethnic, Streetwear, etc.).
3. (Optional) Input weather details to get season-appropriate recommendations.
4. Click "Get Outfit Recommendation" to receive AI-powered fashion advice.

### **AI Stylist**
1. Upload an image of your outfit.
2. The AI will analyze the clothing and suggest improvements or complementary pieces.
3. View recommended outfits to match your style.

### **Chatbot**
- The chatbot guides users by asking simple questions like:
  - "Would you like recommendations for Men or Women?"
  - "What is the occasion?"
  - "What’s the weather like today?"
  - "Would you like to upload an image for styling feedback?"

- **Chatbot Not Responding**: Verify Google AI Studio integration.
- **Streamlit App Crashing**: Check if all required Python packages are installed.
